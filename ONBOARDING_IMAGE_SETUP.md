# Настройка фонового изображения для онбординга

## Что нужно сделать

Добавить изображение **Rectangle 24** (зеленые холмы с оранжевыми цветами) в качестве фона для страницы онбординга.

## Инструкция

1. **Экспортируйте Rectangle 24 из Figma:**
   - Откройте макет в Figma
   - Найдите слой "Rectangle 24" с изображением зеленых холмов
   - Экспортируйте как JPG (качество 90%) или PNG
   - Рекомендуемые размеры: 1512x982 px (или больше для retina)

2. **Добавьте файл в проект:**
   ```bash
   # Поместите файл в папку public с именем:
   /public/rectangle-24.jpg
   # или
   /public/onboarding-background.jpg
   ```

3. **Обновите код в src/app/onboarding/page.tsx:**
   
   Найдите строку 141-147 и раскомментируйте Image компонент:
   
   ```tsx
   // Было (закомментированное):
   {/* <Image
     src="/rectangle-24.jpg"
     alt="Onboarding background"
     fill
     className="object-cover object-center"
     priority
   /> */}
   
   // Станет (раскомментированное):
   <Image
     src="/rectangle-24.jpg"
     alt="Onboarding background"
     fill
     className="object-cover object-center"
     priority
   />
   ```

4. **Удалите временный градиент:**
   
   Удалите или закомментируйте строку 150:
   ```tsx
   {/* Временный градиент похожий на макеты с зелеными холмами */}
   <div className="absolute inset-0 bg-gradient-to-br from-[#A8D5BA] via-[#7CB985] to-[#96C56E]" />
   ```

## Текущее состояние

Сейчас используется **временный зеленый градиент**, который примерно соответствует цветам с макета. Когда вы добавите настоящее изображение Rectangle 24, страница будет выглядеть точно как на макетах.

## Альтернативный вариант

Если у вас есть изображение под другим именем, просто измените путь в `src`:
```tsx
src="/your-image-name.jpg"
```
